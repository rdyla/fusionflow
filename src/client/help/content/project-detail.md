## Project detail

Everything for a single implementation lives here, organized into tabs.

- **Overview** — key dates, team, and status at a glance.
- **Tasks / Timeline** — the work plan; tasks roll up into stages and phases.
- **Documents / SharePoint** — files for the project.
- **CRM Case** — hours accounting against the opportunity SOW.

On phase-scoped projects, each phase tracks its own case and SharePoint folder. If a tab looks empty or wrong, send a request with **Still need help?** and include the project name.
